## 功能

- [x] 获得书籍信息
- [x] 批量下载图片
- [ ] 打包发布
- [ ] 图片合成 PDF
- [ ] 断点续传
- [ ] 多浏览器支持
- [ ] chromedriver 自动下载

## 执行

安装 Chrome 浏览器，并下载对应自己浏览器版本的 chromedriver.exe 放在 python 目录下。

- Chrome 版本：在地址栏输入`chrome://version/`，第一行
- 版本对应下载：https://npm.taobao.org/mirrors/chromedriver/

安装 Python 包：

```bash
pip install -r requirements.txt
```

执行：

```bash
python gui.py
```
